## 5. Особенности BP-06 (Клиентская поддержка)

BP-06 отличается от других процессов: он **оказывает услуги** другим департаментам, а не требует их участия.

| Роль | Департамент | Что происходит |
|------|-------------|----------------|
| **◆ Владелец** | L1-06 Клиент. сервис | Принимает звонки, маршрутизирует, 24/7 поддержка |
| **● Участник** | L1-07 IT | Обеспечивает телефонию, CRM, интеграции |
| **○ Получатель** | L1-01 Агентства | Получает маршрутизацию на КАМов |
| **○ Получатель** | L1-02 Опер. доходы | Получает ошибки для исправления |
| **○ Получатель** | L1-03 MICE | Получает 24/7 поддержку для корп. клиентов |
| **○ Получатель** | L1-04 HR | Получает данные качества звонков |
| **○ Получатель** | L1-05 Операционный | Получает маршрутизацию заявок |
| **○ Получатель** | L1-08 Финансы | Получает заявления на перенос/возврат |

## 7. Почему L1-01 Агентства не имеет отдельного БП

| Аргумент | Пояснение |
|----------|-----------|
| **Не создаёт уникального потока** | Агенты продают те же туры через тот же Lead-to-Cash |
| **Канал, а не процесс** | Агентская сеть — канал дистрибуции, как сайт или телефон |
| **Интегрирован в BP-01** | Агентский кабинет — один из входов в процесс обработки заявок |
| **Обслуживается BP-02** | КАМы привлекают агентов через маркетинговые мероприятия |
| **Обслуживается BP-06** | Регистрация агентов, маршрутизация обращений |

---

## 8. Почему L1-04 HR участвует только в 1 процессе

| Аспект | Пояснение |
|--------|-----------|
| **Владеет BP-07** | HR — единственный процесс управления персоналом |
| **Не участвует активно** | HR не выполняет действия в продажах, маркетинге, финансах |
| **Влияет косвенно** | Через качество персонала HR влияет на ВСЕ процессы |
| **Получает данные** | Из BP-06 получает данные качества звонков для обучения |
| **Получает услуги** | От BP-08 (IT) — учётные записи при найме/увольнении |

> **Вывод:** HR — «сервисный» департамент, который не участвует в операционных процессах, но обеспечивает их кадрами.
