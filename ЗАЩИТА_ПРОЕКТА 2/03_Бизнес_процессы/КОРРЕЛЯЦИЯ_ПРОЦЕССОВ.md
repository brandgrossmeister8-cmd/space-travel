## 2. КЛЮЧЕВЫЕ ТОЧКИ ПЕРЕСЕЧЕНИЯ ПРОЦЕССОВ

### 2.1. Цикл обработки заявки (Lead-to-Cash)

**Участники:** 5 департаментов
**Критичность:** Высокая (85% выручки)

**Схема потока:**

| ЭТАП 1 | → | ЭТАП 2 | → | ЭТАП 3 |
|:------:|:-:|:------:|:-:|:------:|
| **L1-06 Клиентский сервис** | | **L1-05 Операционный** | | **L1-08 Финансы** |
| Приём обращения | → | Расчёт и бронирование | → | Выставление счёта |
| Маршрутизация | | Контроль тайм-лимитов | | Приём оплаты |
| Регистрация агента | | Выписка документов | | Разнесение платежа |
| | | ↑ | | |
| | | **L1-02 Опер. доходы** | | |
| | | Ценообразование | | |
| | | Тарификация | | |

| ЭТАП 4 | ← | СВЯЗЬ |
|:------:|:-:|:-----:|
| **L1-01 Агентства** | ← | Контроль задолженности (от L1-08) |
| Сопровождение КАМом | | |

**Детализация этапов:**

| Этап | Департамент | Функция | Передаёт → |
|------|-------------|---------|------------|
| 1 | L1-06 Клиентский сервис | Приём обращения, маршрутизация, регистрация агента | → L1-05 |
| 2 | L1-05 Операционный | Расчёт, бронирование, контроль тайм-лимитов, выписка | → L1-08 |
| 3 | L1-02 Опер. доходы | Ценообразование, тарификация (параллельно) | → L1-05 |
| 4 | L1-08 Финансы | Выставление счёта, приём оплаты, разнесение платежа | → L1-01 |
| 5 | L1-01 Агентства | Сопровождение КАМом, контроль задолженности | ← L1-08 |

**Узкие места:**
- Перевод звонков из КЦ в отдел продаж (очередь)
- Ручная маршрутизация по 28 направлениям
- Контроль тайм-лимитов авиабилетов (~15% просрочено)

### 2.3. Корпоративный сегмент (MICE)

**Участники:** 5 департаментов
**Критичность:** Высокая (10% выручки, растущий сегмент)

| Этап | Департамент | Функция | Передаёт → |
|------|-------------|---------|------------|

| 3 | L1-08 Бухгалтерия | Счета, сверки, оплаты партнёрам | → L1-03 |
| 4 | L1-09 Контент-маркетинг | B2B-рассылки, лендинги | → L1-03 |
| 5 | L1-06 Отдел 24/7 | Поддержка MICE-клиентов в поездке | → L1-03 |

**Узкие места:**
- Трёхсторонняя координация (перевозчик ↔ ETM ↔ Space Travel)
- Ручное составление смет

### 2.4. Финансовый цикл

**Участники:** 3 департамента
**Критичность:** Критическая (100% денежных потоков)

| Этап | Департамент | Функция | Передаёт → |

| 4 | L1-01 КАМы | Кредитные лимиты, контроль задолженностей | ← L1-08 |

**Узкие места:**
- Нет интеграции GDS → 1С (ручной учёт авиабилетов)
- Invoice Tracker на Google Таблицах (сбои)
- 12 шагов на формирование P&L

### 2.5. Поддержка клиентов и агентов

**Участники:** 3 департамента + внешние партнёры
**Критичность:** Высокая

| Этап | Департамент | Функция | Передаёт → |
|------|-------------|---------|------------|
| 1 | L1-06 Отдел КЦ | Приём звонков, маршрутизация | → L1-05 (очередь!) |
| 2 | L1-05 Отдел продаж | Обработка заявок, расчёты | → L1-06 (статус) |
| 3 | L1-06 Отдел 24/7 | Экстренная помощь, поддержка в туре | ↔ DMC-партнёры |
| 4 | L1-01 КАМы | Обратная связь от агентов | ← L1-06 |

**Узкие места:**
- Перегрузка отдела продаж — звонки не переводятся
- Непрофильный функционал в КЦ (договоры, ГП)
