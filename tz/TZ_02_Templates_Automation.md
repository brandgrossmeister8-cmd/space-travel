## 2. ПРОБЛЕМА (AS-IS)

**Суть проблемы:**
> Сотрудники ежедневно тратят 5.5 часов на создание документов с нуля: сметы в Excel, коммерческие предложения вручную, задачи на мероприятия, типовые ответы клиентам — хотя 80% этих документов повторяются.

**Как сейчас работает процесс:**

| Документ | Что делает сотрудник | Сколько времени |
|----------|----------------------|-----------------|
| Смета MICE | Открывает Excel → копирует прошлую смету → вручную меняет позиции, цены → пересчитывает формулы → оформляет → экспортирует PDF | 40-60 мин на 1 смету |
| КП на тур | Открывает Word/Excel → копирует данные из САМО → подставляет цены → добавляет описания отелей → оформляет | 30-40 мин на 1 КП |
| Задачи на мероприятие | Создаёт задачи в Битрикс24 вручную → назначает ответственных → ставит сроки → копирует чек-листы из прошлых проектов | 20-30 мин на мероприятие |
| Ответ клиенту | Набирает типовой текст заново → копирует из прошлых писем → ищет нужные формулировки | 5-10 мин на письмо |

**Боли:**

| Проблема | К чему приводит | Как часто |
|----------|-----------------|-----------|
| Ошибки в расчётах смет | Неправильная цена клиенту → убытки или потеря сделки | 2-3 раза/неделю |
| Устаревшие цены в КП | Менеджер берёт данные из старого файла | 5-7 раз/неделю |
| Забытые задачи на мероприятие | Не сделан заказ кейтеринга / не забронирован зал | 1-2 раза/месяц |
| Разнобой в ответах клиентам | Разные сотрудники отвечают по-разному | Постоянно |
| Время на рутину вместо продаж | Менеджер занят документами, а не клиентами | Ежедневно |

**Затраты времени сейчас:**

| Операция | Время | Кто делает | Частота |
|----------|-------|------------|---------|
| Сметы MICE | 120 мин/день | Менеджеры MICE (3 чел.) | ~3 сметы/день |
| КП из САМО | 120 мин/день | Менеджеры операционного (10 чел.) | ~4 КП/день |
| Шаблоны задач | 60 мин/день | Менеджеры MICE + маркетинг | ~2 мероприятия/день |
| Типовые ответы Outlook | 30 мин/день | Клиентский сервис (12 чел.) | ~10-15 писем/день |
| **ИТОГО** | **330 мин/день (5.5 ч)** | | |

## 3. ЦЕЛЬ (TO-BE)

**Что хотим получить:**
> Сотрудник выбирает шаблон → система автоматически подставляет данные → документ готов за 5-10 минут вместо 30-60.

**Как будет работать процесс:**

| Документ | Как будет | Время |
|----------|-----------|-------|
| Смета MICE | Выбрать клиента → отметить услуги в каталоге → цены подтянутся автоматически → PDF одним кликом | 10-15 мин |
| КП на тур | В карточке заявки САМО нажать «Создать КП» → данные подставятся → выбрать шаблон → готово | 5-10 мин |

### 7.2. Функции

#### F-01: Конфигуратор смет MICE

| Параметр | Значение |
|----------|----------|
| **Приоритет** | Must |
| **Для кого** | Менеджер MICE |
| **AUTO-точка** | AUTO-013 |

**Что делает:**
> Как менеджер MICE, я хочу создавать сметы выбором услуг из каталога, чтобы не считать вручную и не ошибаться в ценах.

**Сценарий работы:**
1. Менеджер открывает «Создать смету» в Битрикс24
2. Выбирает клиента (из CRM) или вводит нового
3. Указывает параметры: даты, количество участников, город
4. Отмечает услуги из каталога (галочки):
   - Площадка (список с ценами)
   - Питание (завтрак/обед/ужин/кофе-брейк)
   - Оборудование (проектор, экран, микрофон)
   - Трансфер (тип, количество)
   - Проживание (отель, категория)
5. Система автоматически считает:
   - Стоимость каждой позиции = цена × кол-во × дни
   - Итого по категориям
   - Общая сумма
   - Маржа (если настроено)
6. Менеджер нажимает «Сформировать PDF»
7. Система генерирует брендированный документ
8. **Итог:** Смета готова за 10-15 минут

**Каталог услуг (структура):**

| Категория | Примеры позиций | Ед. изм. | Базовая цена |
|-----------|-----------------|----------|--------------|
| Площадки | Конференц-зал (до 50 чел.), Банкетный зал | час/день | 5 000 — 50 000 ₽ |
| Питание | Кофе-брейк, Бизнес-ланч, Гала-ужин | чел. | 500 — 5 000 ₽ |
| Оборудование | Проектор, Экран, Флипчарт, Микрофон | шт./день | 1 000 — 10 000 ₽ |
| Трансфер | Микроавтобус, Автобус, VIP-авто | рейс | 3 000 — 30 000 ₽ |
| Проживание | Стандарт, Бизнес, Люкс | ночь | 4 000 — 25 000 ₽ |
| Доп. услуги | Синхронный перевод, Фотограф, Ведущий | услуга | 10 000 — 100 000 ₽ |

**Альтернативные сценарии:**

| Ситуация | Что делает система |
|----------|---------------------|
| Услуги нет в каталоге | Позволить ввести вручную (название, цена) |
| Нужна скидка | Поле «Скидка %» для всей сметы или по позициям |
| Несколько вариантов | Создать несколько смет → сравнительная таблица |

**Критерии приёмки:**
- [ ] Каталог содержит минимум 50 позиций (по данным от Питуркина)
- [ ] Автоматический расчёт: сумма = цена × кол-во × дни × участники
- [ ] Генерация PDF с логотипом, реквизитами, датой
- [ ] Сохранение истории смет в карточке клиента
- [ ] Время создания сметы: < 15 минут (замер)

#### F-02: Шаблоны КП из САМО

| Параметр | Значение |
|----------|----------|
| **Приоритет** | Must |
| **Для кого** | Менеджер операционного отдела |
| **AUTO-точка** | AUTO-025 |

**Что делает:**
> Как менеджер, я хочу создавать коммерческое предложение из карточки заявки в САМО одним кликом, чтобы клиент быстро получил красивый документ с актуальными данными.

**Сценарий работы:**
1. Менеджер открывает заявку в САМО-тур
2. Нажимает кнопку «Создать КП» (или в контекстном меню)
3. Система подтягивает данные заявки:
   - Направление, отель, категория номера
   - Даты заезда/выезда, ночей
   - Питание
   - Перелёт (рейсы, время)
   - Стоимость (взрослые, дети)
4. Менеджер выбирает шаблон:
   - Стандартный (текст + таблица)
   - Премиум (с фото отеля)
   - Краткий (только цены)
5. Менеджер может:
   - Добавить комментарий
   - Изменить фото
   - Скорректировать описание
6. Нажимает «Сформировать»
7. Система генерирует PDF
8. **Итог:** КП готово за 5-10 минут

**Данные для подстановки:**

| Поле из САМО | Поле в КП |
|--------------|-----------|
| Направление | «Турция, Анталья» |
| Отель | «Rixos Premium Belek 5*» |
| Номер | «Deluxe Room, вид на море» |
| Даты | «15.06.2026 — 22.06.2026 (7 ночей)» |
| Питание | «Ultra All Inclusive» |
| Перелёт | «SU 2134, Москва-Анталья, 10:00-13:30» |
| Стоимость | «245 000 ₽ за 2 взрослых» |

**Альтернативные сценарии:**

| Ситуация | Что делает система |
|----------|---------------------|
| Несколько вариантов туров | Выбрать несколько заявок → сравнительное КП |
| Данные неполные | Подсветить пустые поля, не блокировать генерацию |
| Нужно изменить данные | Позволить редактировать перед генерацией |

**Критерии приёмки:**
- [ ] Кнопка «Создать КП» в карточке заявки САМО
- [ ] Минимум 3 шаблона КП (стандарт, премиум, краткий)
- [ ] Автоподстановка всех данных из заявки
- [ ] Генерация PDF с логотипом компании
- [ ] Время создания КП: < 10 минут (замер)

#### F-03: Шаблоны проектов и задач Битрикс24

| Параметр | Значение |
|----------|----------|
| **Приоритет** | Must |
| **Для кого** | Менеджер MICE, маркетолог |
| **AUTO-точка** | AUTO-027 |

**Что делает:**
> Как менеджер MICE, я хочу создавать проект мероприятия из готового шаблона, чтобы не забыть ни одну задачу и не настраивать сроки вручную.

**Сценарий работы:**
1. Менеджер нажимает «Создать проект» в Битрикс24
2. Выбирает тип мероприятия:
   - Бизнес-завтрак (10-30 чел.)
   - Конференция (50-200 чел.)
   - Корпоратив (50-500 чел.)
   - Вебинар
   - Выставка/стенд
3. Указывает:
   - Название мероприятия
   - Дату проведения
   - Клиента (из CRM)
4. Система создаёт проект с задачами:
   - Все задачи из шаблона
   - Сроки = «дата мероприятия минус X дней»
   - Чек-листы внутри задач
   - Ответственные по умолчанию
5. **Итог:** Проект с 15-30 задачами создан за 5 минут

**Пример шаблона «Бизнес-завтрак»:**

| Задача | Срок (до мероприятия) | Ответственный |
|--------|----------------------|---------------|
| Согласовать концепцию с клиентом | -30 дней | Менеджер MICE |
| Забронировать площадку | -25 дней | Менеджер MICE |
| Составить смету | -25 дней | Менеджер MICE |
| Согласовать смету с клиентом | -20 дней | Менеджер MICE |
| Получить предоплату | -14 дней | Финансы |
| Заказать кейтеринг | -10 дней | Менеджер MICE |
| Подготовить раздаточные материалы | -7 дней | Маркетинг |
| Подтвердить участников | -3 дня | Менеджер MICE |
| Провести мероприятие | 0 дней | Менеджер MICE |
| Отправить благодарственные письма | +1 день | Менеджер MICE |
| Собрать обратную связь | +3 дня | Менеджер MICE |
| Подготовить отчёт | +5 дней | Менеджер MICE |

**Альтернативные сценарии:**

| Ситуация | Что делает система |
|----------|---------------------|
| Нужен нестандартный тип | Создать пустой проект + выбрать задачи из библиотеки |
| Нужно изменить сроки | Все сроки автоматически пересчитываются от даты мероприятия |
| Несколько ответственных | Указать команду проекта при создании |

**Критерии приёмки:**
- [ ] Минимум 5 шаблонов проектов (по списку выше)
- [ ] Каждый шаблон содержит 10-30 задач
- [ ] Автоматический расчёт сроков от даты мероприятия
- [ ] Чек-листы внутри задач (где нужно)
- [ ] Время создания проекта: < 5 минут

#### F-04: Библиотека Quick Parts Outlook

| Параметр | Значение |
|----------|----------|
| **Приоритет** | Should |
| **Для кого** | Сотрудник клиентского сервиса |
| **AUTO-точка** | AUTO-036 |

**Что делает:**
> Как сотрудник клиентского сервиса, я хочу вставлять типовые ответы горячей клавишей, чтобы не набирать одно и то же каждый раз.

**Сценарий работы:**
1. Сотрудник пишет письмо клиенту

5. Текст вставляется в письмо
6. Сотрудник персонализирует (имя клиента, номер заявки)

**Категории блоков:**

| Категория | Примеры блоков |

| Завершение | «С уважением», «Хорошего отдыха», «Ждём отзыв» |

**Критерии приёмки:**
- [ ] Минимум 30 блоков текста (по списку от Акоповой)
- [ ] Блоки сгруппированы по категориям
- [ ] Горячая клавиша для быстрого доступа
- [ ] Инструкция для сотрудников
- [ ] Возможность добавлять свои блоки
