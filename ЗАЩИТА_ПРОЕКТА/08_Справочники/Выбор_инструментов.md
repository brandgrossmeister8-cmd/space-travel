## Что такое инструменты автоматизации

### Простое объяснение

Представьте, что у вас есть сотрудник, который каждый день:
- Копирует данные из одной программы в другую
- Проверяет сроки и отправляет напоминания
- Собирает отчёты из разных источников
- Отвечает на одинаковые вопросы клиентов

**Инструменты автоматизации — это программы, которые делают эту работу вместо человека.**

Они не заменяют сотрудников, а освобождают их от рутины для более важных задач: общения с клиентами, решения сложных ситуаций, развития бизнеса.

### Почему это важно для Space Travel

По результатам анализа (Этап 3), ваши сотрудники тратят **~1 966 часов в месяц** на задачи, которые можно автоматизировать. Это эквивалентно **~11 сотрудникам**, работающим полный день только на рутине (1 966 ч ÷ 176 ч/мес = 11.2).

**Потенциальная экономия:**
- **NET-экономия времени:** 1 358 ч/мес (~7,7 сотрудников)
- **В деньгах:** ~14,8 млн ₽/год (при дифференцированных ставках L1/L2/L3)

## Категория 1: RPA — роботизация рутины

### Что это такое (простым языком)

**RPA** (Robotic Process Automation — «роботизация процессов») — это программа-робот, которая работает как виртуальный сотрудник. Она умеет:

- **Нажимать кнопки** в программах (как человек мышкой)
- **Заполнять формы** — переносить данные из одного окна в другое
- **Читать экран** — понимать, что написано на экране, и принимать решения
- **Работать с файлами** — открывать Excel, PDF, копировать текст

### Когда это нужно

RPA подходит, когда:
- Нужно работать со **старыми программами**, у которых нет современных способов обмена данными
- Сотрудник выполняет **одинаковые действия** много раз в день
- Задача требует работы в **нескольких окнах** одновременно

### Примеры для Space Travel

| Точка автоматизации | Как RPA решает задачу |
|---------------------|----------------------|
| AUTO-005: GDS → 1С (учёт авиабилетов) | Робот сам заходит в GDS, копирует данные о выписанных билетах и вносит их в 1С |
| AUTO-010: Сверка пассажиров САМО ↔ GDS | Робот сравнивает списки в двух программах и отмечает расхождения |
| AUTO-032: Категоризация расходов в 1С | Робот читает назначение платежа и автоматически присваивает категорию |

### Сравнение инструментов RPA

| Критерий | UiPath | Microsoft Power Automate Desktop | Электронные ассистенты |
|----------|--------|----------------------------------|------------------------|
| **Что это** | Мировой лидер RPA, специализированная платформа | Часть пакета Microsoft 365 | Российские решения (Primo, ROBIN) |
| **Сложность** | Требует обучения, 2-4 недели | Проще в освоении, 1-2 недели | Сопоставимо с UiPath |
| **Стоимость** | от $420/мес за робота (~50 000 ₽/мес) | Бесплатно в Microsoft 365 E3/E5, или ~$15/мес | от 30 000 ₽/мес |
| **Плюсы** | Мощный, много готовых модулей | Уже есть если есть Microsoft 365 | Русскоязычная поддержка, работа с 1С |
| **Минусы** | Дорого, избыточно для малых задач | Слабее для сложных сценариев | Меньше сообщество, меньше обучающих материалов |
| **Подходит для** | AUTO-005, AUTO-015, AUTO-019 (сложные интеграции) | AUTO-010, AUTO-032, AUTO-033 (средние задачи) | AUTO-005 (если критична работа с 1С) |

### Рекомендация для Space Travel

**Основной инструмент:** Microsoft Power Automate Desktop
**Почему:** У вас уже используется Outlook (часть Microsoft 365), поэтому Power Automate будет бесплатным или дешёвым дополнением. Для 80% задач его достаточно.

**Для сложных задач:** Рассмотреть российские решения (Primo RPA) для интеграции GDS → 1С (AUTO-005), так как это критически важная точка с экономией 81 ч/мес.

## Категория 2: iPaaS — интеграция систем

### Что это такое (простым языком)

**iPaaS** (Integration Platform as a Service — «платформа интеграции как сервис») — это сервис, который **соединяет разные программы между собой** без программирования.

**Простая аналогия:** Представьте, что у вас есть 5 разных приложений на телефоне. iPaaS — это «переводчик», который позволяет им общаться друг с другом.

**Пример:** Когда в САМО-тур появляется новая заявка → автоматически создаётся задача в Битрикс24 → отправляется письмо клиенту через Outlook → данные попадают в аналитику RoiStat.

### Ключевые понятия

| Термин | Что это значит |
|--------|----------------|
| **API** | «Язык общения» программ. Если у программы есть API — её можно подключить к iPaaS |
| **Триггер** | Событие, которое запускает автоматизацию («новая заявка», «оплата получена») |
| **Сценарий** | Последовательность действий: «когда произошло X → сделай Y → потом Z» |
| **Коннектор** | Готовое подключение к популярной программе (Битрикс24, Outlook, Telegram) |

### Когда это нужно

iPaaS подходит, когда:
- Нужно **передавать данные** между программами автоматически
- Программы имеют **API** (современный способ обмена данными)
- Сценарии относительно **простые**: «если X — делай Y»

### Примеры для Space Travel

| Точка автоматизации | Как iPaaS решает задачу |
|---------------------|------------------------|
| AUTO-011: САМО → Email-рассылки | Когда в САМО появляется новый тур — автоматически отправляется письмо через Юнисендер с ценами |
| AUTO-024: Мессенджеры → CRM | Сообщения из Telegram/WhatsApp автоматически создают лид в Битрикс24 |
| AUTO-043: Уведомления агентам | Изменение статуса заявки → автоматическое уведомление агенту по email или в Telegram |
| AUTO-039: Отправка документов | После оплаты → автоматическая отправка ваучера и договора клиенту |

### Сравнение инструментов iPaaS

| Критерий | Albato | Make (ex-Integromat) | Zapier | n8n |
|----------|--------|----------------------|--------|-----|
| **Что это** | Российский сервис интеграций | Международный, популярный в Европе | Мировой лидер | Open-source (бесплатный) |
| **Русскоязычные системы** | ✅ Битрикс24, 1С, amoCRM | ⚠️ Частично | ❌ Слабо | ⚠️ Через настройку |
| **САМО-тур** | ⚠️ Через API/Webhook | ⚠️ Через API/Webhook | ❌ Нет коннектора | ⚠️ Через API |
| **Стоимость** | от 990 ₽/мес (299 задач) | от $9/мес (1000 операций) | от $19/мес (750 задач) | Бесплатно (самостоятельно) |
| **Простота** | ⭐⭐⭐⭐⭐ Очень просто | ⭐⭐⭐⭐ Средне | ⭐⭐⭐⭐⭐ Очень просто | ⭐⭐⭐ Требует IT-навыков |
| **Плюсы** | Русская поддержка, знает 1С | Гибкий, мощный | Много коннекторов | Бесплатно, на своём сервере |
| **Минусы** | Меньше международных интеграций | Нет прямой поддержки 1С | Дорого, слабо с 1С | Нужен IT-специалист |

### Рекомендация для Space Travel

**Основной инструмент:** Albato
**Почему:**
1. Русскоязычная поддержка и интерфейс
2. Готовые коннекторы для Битрикс24, 1С, Юнисендер, Telegram
3. Простота: сотрудники могут создавать интеграции сами после 1-дневного обучения
4. Цена: от 990 ₽/мес — дешевле международных аналогов

**Для сложных сценариев:** Make (Integromat) — если нужна сложная логика с ветвлениями и циклами.

**Бесплатная альтернатива:** n8n — если IT-отдел готов поддерживать собственный сервер.

## Категория 3: Low-code — создание приложений без программирования

### Что это такое (простым языком)

**Low-code** (дословно «мало кода») — это платформы, которые позволяют **создавать программы без программирования**. Вы собираете приложение как конструктор: перетаскиваете элементы, настраиваете связи, получаете работающую систему.

**Простая аналогия:** Раньше, чтобы построить сайт, нужен был программист. Сейчас есть Tilda — собираете сайт из готовых блоков мышкой. Low-code — то же самое, но для бизнес-приложений.

### Что можно создать на Low-code

- **Формы и анкеты** — сбор данных от клиентов или сотрудников
- **Базы данных** — реестры, справочники, каталоги
- **Рабочие процессы (workflow)** — согласования, маршрутизация заявок
- **Порталы** — личные кабинеты клиентов или агентов
- **Мобильные приложения** — простые приложения для телефона

### Когда это нужно

Low-code подходит, когда:
- Нужно **заменить Excel-таблицы** на нормальную систему
- Требуется **быстро собрать решение** (дни, а не месяцы)
- Нет бюджета на полноценную разработку
- Нужно **часто менять** логику работы

### Примеры для Space Travel

| Точка автоматизации | Как Low-code решает задачу |
|---------------------|---------------------------|
| AUTO-006: Замена Invoice Tracker | Вместо Google Таблиц — приложение с формами, уведомлениями, правами доступа |
| AUTO-009: Маршрутизация заявок | Система автоматического распределения: заявка попадает нужному менеджеру по правилам |
| AUTO-013: Шаблоны смет MICE | Конструктор смет: выбираешь услуги → автоматически считается цена |
| AUTO-041: Согласования в Битрикс24 | Workflow: документ → руководитель отдела → финансы → директор |

### Сравнение инструментов Low-code

| Критерий | Битрикс24 | Microsoft Power Apps | 1С-Битрикс24 + Мой склад | Notion/Airtable |
|----------|-----------|---------------------|--------------------------|-----------------|
| **Что это** | CRM + конструктор процессов | Конструктор приложений от Microsoft | Отраслевые решения | Базы данных + автоматизация |
| **Уже есть у вас** | ✅ Да | ⚠️ Нужен Microsoft 365 | ❌ Нет | ❌ Нет |
| **Сложность** | ⭐⭐⭐ Средне | ⭐⭐⭐⭐ Выше среднего | ⭐⭐⭐ Средне | ⭐⭐ Просто |
| **Стоимость** | Уже оплачено (есть) | от $5/польз/мес | от 15 000 ₽/мес | от $10/польз/мес |
| **Возможности CRM** | ✅ Полноценная CRM | ⚠️ Частично | ✅ Да | ❌ Нет |
| **Интеграция с 1С** | ⚠️ Через API/коннекторы | ⚠️ Сложно | ✅ Да | ❌ Нет |
| **Плюсы** | Всё в одном месте, сотрудники знают | Мощный конструктор | Заточен под торговлю | Очень простой старт |
| **Минусы** | Ограничения кастомизации | Требует обучения | Избыточен для туризма | Не заменит CRM |

### Рекомендация для Space Travel

**Основной инструмент:** Битрикс24 (уже есть)
**Почему:**
1. Уже используется в компании — не нужно обучать сотрудников новой системе
2. Достаточно мощный для 90% задач (workflow, формы, отчёты)
3. Нет дополнительных затрат на лицензии
4. Интеграция с Outlook, телефонией, мессенджерами

**Что доработать в Битрикс24:**
- Настроить воронки продаж (для AUTO-009)
- Создать workflow согласований (для AUTO-041)
- Добавить смарт-процессы для MICE (для AUTO-013)

**Для замены Invoice Tracker (AUTO-006):** Специальное решение внутри Битрикс24 или простое приложение на Power Apps (если есть Microsoft 365 E3/E5).

## Категория 4: BI — бизнес-аналитика и дашборды

### Что это такое (простым языком)

**BI** (Business Intelligence — «бизнес-интеллект») — это программы для **визуализации данных**. Они превращают таблицы с цифрами в понятные графики, диаграммы и дашборды.

**Простая аналогия:** Excel умеет строить графики, но BI делает это в 10 раз красивее, автоматически обновляется и позволяет «провалиться» в детали одним кликом.

### Ключевые понятия

| Термин | Что это значит |
|--------|----------------|
| **Дашборд** | Экран с несколькими графиками и показателями, обновляется автоматически |
| **KPI** | Ключевой показатель эффективности — цифра, которую отслеживаем (выручка, конверсия, NPS) |
| **Drill-down** | «Провалиться в детали» — клик на график показывает подробности |
| **ETL** | Процесс загрузки данных из источников в BI-систему |

### Когда это нужно

BI подходит, когда:
- Руководители тратят **часы на сбор отчётов** вручную
- Данные лежат в **разных системах** (САМО, 1С, Битрикс24)
- Нужны **интерактивные отчёты** с фильтрами и детализацией
- Важно видеть **актуальную картину** (не вчерашнюю, а сегодняшнюю)

### Примеры для Space Travel

| Точка автоматизации | Что покажет BI-дашборд |
|---------------------|------------------------|
| DASH-01: P&L отчётность | Прибыль/убыток по направлениям, менеджерам, периодам — в реальном времени |
| DASH-02: Тайм-лимиты | Сколько билетов «сгорит» сегодня, завтра, на неделе — с алертами |
| DASH-04: Воронка продаж | Сколько заявок на каждом этапе, где «застревают», конверсия |
| DASH-05: Ценообразование | Маржинальность по турам, сравнение с конкурентами |
| DASH-07: Cash Flow | Плановые поступления и платежи, кассовый разрыв |

### Сравнение инструментов BI

| Критерий | Power BI | Yandex DataLens | Metabase | Google Looker Studio |
|----------|----------|-----------------|----------|---------------------|
| **Что это** | Лидер рынка от Microsoft | Российский сервис от Яндекса | Open-source решение | Бесплатный BI от Google |
| **Стоимость** | от $10/польз/мес (~1 200 ₽) | Бесплатно (до 10 GB) | Бесплатно | Бесплатно |
| **Интеграция с 1С** | ✅ Через коннекторы | ⚠️ Через выгрузку | ⚠️ Через базу данных | ⚠️ Через выгрузку |
| **Интеграция с САМО** | ⚠️ Через API/выгрузку | ⚠️ Через выгрузку | ⚠️ Через базу данных | ⚠️ Через выгрузку |
| **Сложность** | ⭐⭐⭐ Средне (1-2 недели обучения) | ⭐⭐ Просто | ⭐⭐⭐ Средне | ⭐⭐ Просто |
| **Возможности** | ⭐⭐⭐⭐⭐ Максимальные | ⭐⭐⭐ Хорошие | ⭐⭐⭐⭐ Очень хорошие | ⭐⭐⭐ Хорошие |
| **Плюсы** | Мощный, много обучения, интеграция с Excel | Бесплатно, русский, простой | Бесплатно, гибкий | Бесплатно, прост |
| **Минусы** | Платный, требует обучения | Меньше функций | Нужен IT для настройки | Ограничен для сложных задач |

### Рекомендация для Space Travel

**Основной инструмент:** Power BI
**Почему:**
1. Лучшая интеграция с Excel (который уже используется для отчётов)
2. Много готовых шаблонов и обучающих материалов
3. Интеграция с Microsoft 365 (Outlook уже есть)
4. Мощные возможности для сложных дашбордов (P&L, Cash Flow)

**Стоимость для Space Travel:**
- 10-15 лицензий Power BI Pro = 12 000–18 000 ₽/мес
- Окупаемость: DASH-01 экономит ~55 ч/мес директора финансов = ~165 000 ₽/мес (при ставке 3 000 ₽/ч)

**Бесплатная альтернатива для простых задач:** Yandex DataLens — для дашбордов, которые не требуют сложной логики.

## Категория 5: Чат-боты — автоматические помощники

### Что это такое (простым языком)

**Чат-бот** — это программа, которая **отвечает на сообщения автоматически**. Она может:
- Отвечать на частые вопросы (FAQ)
- Собирать информацию от клиента (имя, телефон, запрос)
- Отправлять уведомления и напоминания
- Записывать на консультацию или встречу

**Простая аналогия:** Автоответчик, но умный — понимает вопросы и даёт полезные ответы.

### Ключевые понятия

| Термин | Что это значит |
|--------|----------------|
| **Сценарий** | Последовательность сообщений и вопросов, которые задаёт бот |
| **Триггер** | Слово или фраза, которая запускает определённый сценарий |
| **Интеграция с CRM** | Бот передаёт данные в Битрикс24 (создаёт лид, контакт, задачу) |
| **NLP** | Технология понимания естественного языка — бот понимает вопрос, даже если он задан не по шаблону |

### Когда это нужно

Чат-боты подходят, когда:
- Много **повторяющихся вопросов** от клиентов или агентов
- Нужен **быстрый ответ** 24/7 (ночью, в выходные)
- Требуется **первичная квалификация** — понять, чего хочет клиент
- Важно **не терять обращения** — бот запишет контакт, пока менеджер недоступен

### Примеры для Space Travel

## Глоссарий (словарь терминов)

| Термин | Простое объяснение |
|--------|-------------------|
| **API** | «Язык общения» программ между собой. Если программа имеет API — другие программы могут отправлять ей команды и получать данные |
| **BI** | Business Intelligence — программы для красивых отчётов и графиков |
| **Дашборд** | Экран с графиками и цифрами, который обновляется автоматически |
| **ETL** | Процесс загрузки данных из разных источников в одно место для анализа |
| **iPaaS** | Сервис, который соединяет программы между собой без программирования |
| **Low-code** | Платформа, на которой можно собирать приложения как конструктор, без программистов |
| **NLP** | Технология понимания человеческого языка (как Siri или Алиса) |
| **Open-source** | Бесплатная программа с открытым кодом (можно использовать и менять) |
| **ROI** | Return on Investment — окупаемость: сколько денег вернётся на каждый вложенный рубль |
| **RPA** | Программа-робот, которая нажимает кнопки и заполняет формы как человек |
| **SaaS** | Software as a Service — программа по подписке (не покупаете, а арендуете) |
| **Сценарий** | Последовательность действий: «если произошло X — сделай Y» |
| **Триггер** | Событие, которое запускает автоматизацию (новая заявка, оплата, время) |
| **Workflow** | Рабочий процесс — последовательность шагов для выполнения задачи |
