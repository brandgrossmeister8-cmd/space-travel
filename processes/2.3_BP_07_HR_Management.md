---

## Резюме

Процесс управления персоналом обеспечивает все основные процессы компании квалифицированными кадрами. От качества работы HR зависит укомплектованность штата, скорость адаптации новичков, вовлечённость сотрудников и развитие компетенций.

**Влияние на основные процессы:**
- BP-01 (Заявки): Качество менеджеров продаж и бронирования
- BP-02 (Маркетинг): Наличие специалистов по маркетингу
- BP-03 (MICE): Квалификация MICE-менеджеров
- BP-05 (Продукт): Компетенции продуктологов и тарификаторов
- BP-06 (Поддержка): Качество операторов поддержки

---

## Цель процесса

Обеспечить укомплектованность персоналом в соответствии с потребностями бизнеса, снизить текучесть персонала, повысить вовлечённость и развить компетенции сотрудников.

### 7.5.1 Интеграция с контролем качества звонков (AUTO-HR-03)

**Источник данных:** AUTO-SUP-03 (ИИ-анализ качества звонков из BP-06)

#### Входящие данные от BP-06 (Клиентская поддержка)

| Тип данных | Периодичность | Содержание |
|------------|---------------|------------|
| **Еженедельный отчёт** | Каждый понедельник | Сводка оценок по 6 критериям качества |
| **Рейтинг сотрудников** | Еженедельно | Оценки каждого оператора с динамикой |
| **ТОП-3 проблемы** | Еженедельно | Наиболее частые нарушения стандартов |
| **Красные флаги** | Немедленно | Сотрудники с критически низкими оценками (<50%) |
| **Лучшие практики** | Еженедельно | Примеры отличных звонков для обучения |

#### Критерии оценки звонков (из AUTO-SUP-03)

| Критерий | Вес | Фокус обучения при низкой оценке |
|----------|-----|----------------------------------|
| **1. Приветствие** | 10% | Скрипты, стандарты коммуникации |
| **2. Выявление потребности** | 25% | Техники задавания вопросов, активное слушание |
| **3. Компетентность** | 25% | Знание продукта, работа в системах |
| **4. Решение вопроса** | 20% | Принятие решений, эскалация |
| **5. Завершение разговора** | 10% | Стандарты завершения, кросс-продажи |
| **6. Время разговора** | 10% | Тайм-менеджмент, эффективность |

#### Форматы обучения по типам проблем

| Проблема (по критерию) | Формат обучения | Периодичность |

| Выявление потребности | Ролевые игры, разбор кейсов | 2 раза / месяц |
| Компетентность | Продуктовое обучение, тест | По необходимости |
| Решение вопроса | Мастер-класс «Принятие решений» | 1 раз / месяц |
| Время разговора | Индивидуальный коучинг | По необходимости |
