## Выявленные паттерны

### 1. Звёздная топология вокруг OPS

```
         REV ──────┐
                   │
    AGN ──────────►│
                   │
    MICE ─────────►OPS◄────────── SRV
                   │
    HR ───────────►│
                   │
         FIN ──────┘
```

**Вывод:** Операционный департамент — центральный хаб, через который проходит большинство потоков.

### 2. Цепочка Продукт → Тарификация → IT

```
    Отдел продукта ──► Группа тарификации ──► IT
    (задачи)          (расчёт)               (публикация)
```

**Вывод:** Линейная зависимость — задержка на одном этапе блокирует весь поток.

### 3. Веерная маршрутизация в Клиентском сервисе

```
                    ┌── Отдел продаж
                    ├── Отдел бронирования
    Клиент. поддержка ├── VIP-отдел
                    ├── Транспортный
                    ├── Региональные офисы (×8)
                    └── IT, HR, Маркетинг...
```

**Вывод:** Один отдел принимает решения о маршрутизации для ~20 направлений.
