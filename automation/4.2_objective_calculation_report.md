**Дата:** 2026-02-18

**Методология:** Прямой расчёт на основе данных о времени из анкет сотрудников

---

## 1. Источник данных (L3)

### Что такое L3

| Уровень | Кто заполнял | Что содержит |
|---------|--------------|--------------|
| L1 | Директора департаментов (9 чел) | Цели, стратегия, ресурсы |
| L2 | Руководители отделов (37 чел) | Процессы, взаимодействия |
| **L3** | **Сотрудники (150 чел)** | **Время на задачи, частота, уровень рутины** |

### Структура данных L3

Каждый сотрудник указал для своих задач:
- `time_minutes` — время выполнения (минуты)
- `regularity` — частота (ежедневно / еженедельно / ежемесячно / по запросу)
- `routine_level` — уровень рутины (очень высокий / высокий / средний / низкий)
- `task_type` — тип (рутинная / проблемная)

### Формула перевода в часы/месяц

```
часы/мес = минуты × множитель_частоты ÷ 60

Множители:
• Ежедневно      → ×22 (рабочих дней)
• Еженедельно    → ×4.33 (недель в месяце)
• Ежемесячно     → ×1
• По запросу     → ×5 (консервативно)
```

**Пример:** Антошкина, "Корректировки в заявку", 500 мин, ежедневно → 500 × 22 ÷ 60 = **183 ч/мес**

### Что такое FTE

**FTE** (Full-Time Equivalent) = эквивалент полной занятости.

```
1 сотрудник = 22 дня × 8 часов = 176 часов/месяц
FTE = Общие часы ÷ 176

30 593 ÷ 176 = 174 FTE
```
