# Матрица Impact × Effort — Space Travel

**Версия:** 1.0
**Дата:** 2026-02-17
**Источник:** 3.4_automation_potential.md v2.0 (64 точки)

> **Методология:** Impact = экономия ч/мес (данные из реестра). Effort = сложность 1–5.
> Порог квадрантов: High Impact ≥ 20 ч/мес; High Effort = сложность 4–5.
> Дубли AUTO+DASH считаются как один проект (единая экономия).

---

## Четыре квадранта

```

     (делать немедленно)    │         (планировать и обосновывать)
                            │
   Сложность 1–3            │              Сложность 4–5
   ─────────────────────────┼─────────────────────────────────────
   Сложность 1–3            │              Сложность 4–5
                            │
        Q3: FILL-INS        │         Q4: НЕ ДЕЛАТЬ / ОТЛОЖИТЬ
   (делать по возможности)  │         (низкий ROI относительно усилий)
                            │
                   НИЗКАЯ ЭКОНОМИЯ (<20 ч/мес)
```
