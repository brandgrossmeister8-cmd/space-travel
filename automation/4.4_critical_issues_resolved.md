## K1: Назначение и.о. владельца для маркетинговых AUTO-точек

### Проблема
5 AUTO-точек привязаны к L1-09 (Маркетинг), но позиция директора по маркетингу вакантна. Без владельца процесса невозможно:
- Согласовать требования
- Принять результат
- Обеспечить внедрение

### Решение

**И.о. владельца маркетинговых AUTO-точек:** Преображенская Т.В. (операционный директор)

**Обоснование:**
1. Операционный директор курирует клиентский путь, включая маркетинговые касания
2. Исторически маркетинг подчинялся операционному блоку
3. Временное решение до назначения директора по маркетингу

### Распределение ответственности

| AUTO | Название | Было | Стало |
|------|----------|------|-------|
| AUTO-011 | САМО → Email-рассылки | L1-09 (вакансия) | **Преображенская Т.В. (и.о.)** |
| AUTO-026 | ИИ-генерация визуалов | L1-09 (вакансия) | **Преображенская Т.В. (и.о.)** |
| AUTO-028 | Авторассылка напоминаний MICE | L1-09 + L1-03 | **Питуркин С.А. (L1-03)** — основной |
| AUTO-048 | Авторотация рекламы | L1-09 (вакансия) | **Преображенская Т.В. (и.о.)** |
| DASH-06 | RoiStat аналитика | L1-09 (вакансия) | **Преображенская Т.В. (и.о.)** |

### Эскалация при назначении директора по маркетингу
При найме директора по маркетингу:
1. Передать владение AUTO-011, AUTO-026, AUTO-048, DASH-06
2. Провести handover по текущему статусу
3. Пересмотреть приоритеты маркетинговой автоматизации

## K2: Contingency Plan для API САМО-тур

### Проблема
P-01 (аудит API САМО) — критическая зависимость для 12 AUTO-точек. Результат аудита неизвестен. Если API ограничен — 30% roadmap под угрозой.

### Зависимые AUTO-точки (12 шт.)

| AUTO | Название | Зависимость от API | Альтернатива |
|------|----------|-------------------|--------------|
| AUTO-003 | Динамическое ценообразование | **Критическая** (обновление цен) | Файловый экспорт + RPA |
| AUTO-010 | Сверка САМО ↔ GDS | Средняя (чтение данных) | Выгрузка в Excel + скрипт |
| AUTO-011 | САМО → Email | **Критическая** (цены туров) | Ручная выгрузка → Юнисендер |
| AUTO-025 | Шаблоны КП из САМО | Средняя (данные тура) | Ручной ввод в шаблон |
| AUTO-035 | Синхронизация агентов | **Критическая** (двусторонняя) | Односторонняя + ручная сверка |
| AUTO-043 | Уведомления агентам | Средняя (статусы заявок) | Webhook (если есть) |
| AUTO-044 | Контроль лимитов | **Критическая** (блокировка) | Ручной контроль + Excel |
| AUTO-016 | Ревизия туров | Низкая (отчёт) | Встроенный отчёт САМО |
| AUTO-017 | Мониторинг сроков цен | Низкая (алерты) | Встроенные уведомления САМО |
| AUTO-034 | Автопроверка тарификации | Средняя (валидация) | Скрипт на выгрузках |
| TZ-01 | Интеграция САМО ↔ Б24 | **Критическая** | Полуавтоматический режим |
| TZ-06 | Email-рассылки из САМО | **Критическая** | Ручной экспорт цен |

### Сценарии результатов P-01

#### Сценарий A: API полноценный (вероятность 60%)
- Все 12 AUTO-точек реализуются по плану
- Roadmap без изменений
- TZ-01, TZ-06 — полная автоматизация

#### Сценарий B: API ограниченный (вероятность 30%)
**Ограничения:** Только чтение, rate limits, нет webhooks

| Воздействие | Решение |
|-------------|---------|
| Нет webhooks | Polling каждые 5 минут через Albato |
| Rate limits | Кэширование + пакетная обработка |
| Только чтение | Односторонняя синхронизация САМО → Б24 |

**Корректировка roadmap:**
- TZ-01: Односторонняя синхронизация (экономия -30%)
- AUTO-044: Ручная блокировка с алертами (экономия -50%)
- Общее снижение эффекта: ~200 ч/мес → ~140 ч/мес

#### Сценарий C: API отсутствует (вероятность 10%)
**Альтернативный подход:**

| AUTO | Альтернативное решение | Экономия vs план |
|------|------------------------|------------------|
| AUTO-003 | RPA (Primo) для обновления цен | 70% |
| AUTO-011 | Выгрузка CSV → Юнисендер (автоматизация через n8n) | 50% |
| AUTO-035 | RPA для создания агентов в САМО | 60% |
| AUTO-043 | Парсинг email-уведомлений САМО | 40% |
| AUTO-044 | Внешний модуль контроля лимитов | 50% |

**Дополнительные затраты Сценария C:**
- Primo RPA: +150 000 ₽ (лицензия) + 30 000 ₽/мес
- Разработка RPA-сценариев: +400 000 ₽
- Общее увеличение бюджета Волны 1: **+30%**

### Рекомендации по P-01

| # | Действие | Срок | Ответственный |
|---|----------|------|---------------|

- Выбран сценарий (A/B/C)
