## Легенда

| Элемент | Значение |
|---------|----------|
| `═══►` | Автоматическая интеграция (API) |
| `───►` | Полуавтоматическая (файл + обработка) |
| `- - ►` | Ручной перенос данных |
| `┌───┐` | Внутренняя система Space Travel |
| `╔═══╗` | Внешняя система / платформа |
| `🔴` | Критическая проблема интеграции |
| `⚠️` | Проблема средней важности |
