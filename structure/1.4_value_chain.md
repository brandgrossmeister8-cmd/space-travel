## Что такое цепочка создания ценности?

**Цепочка создания ценности** (Value Chain, модель Портера) — это последовательность бизнес-процессов компании, которые создают ценность для клиента и приносят прибыль.

Делится на:
- **Основные активности** — напрямую создают продукт и доставляют его клиенту
- **Поддерживающие активности** — обеспечивают работу основных процессов

## Потоки создания ценности

### Поток 1: B2C через агентства (основной)

```
L1-02 Продукт → L1-05 Операции → L1-01 Агентства → L1-06 Сервис
     │              │                 │                  │
     ▼              ▼                 ▼                  ▼
 Контракты     Бронирование      Агентская сеть      Поддержка
 с отелями     и оформление      2,5+ млрд руб.      клиентов
```

**Доля выручки:** ~85%
**Участники:** 155 чел.

**Как работает:**
1. **Продукт** договаривается с отелями, авиакомпаниями о ценах и квотах
2. **Операции** принимают заказ от агента, бронируют, оформляют документы
3. **Агентства** продают туры конечным туристам через партнёрскую сеть
4. **Сервис** сопровождает туриста до, во время и после поездки

### Поток 2: B2B корпоративный (MICE / Бизнес-тревел)

```
L1-02 Продукт → L1-03 MICE → L1-05 Операции → L1-06 Сервис
     │              │              │                │
     ▼              ▼              ▼                ▼
 Спецпредложения  Тендеры,     Исполнение       VIP-
 для компаний     контракты    заказов          сопровождение
```

**Доля выручки:** ~10%
**Участники:** 89 чел.

**MICE** — Meetings, Incentives, Conferences, Exhibitions:
- **M**eetings — корпоративные встречи
- **I**ncentives — поощрительные поездки для сотрудников
- **C**onferences — конференции
- **E**xhibitions — выставки

### Поток 3: Въездной туризм (Inbound)

```
L1-03 Въездной → L1-05 Операции → L1-06 Сервис
      │               │                │
      ▼               ▼                ▼
  Партнёры        Визовая          Трансферы,
  за рубежом      поддержка        экскурсии
```

**Доля выручки:** ~5%
**Участники:** 68 чел.

**Въездной туризм** — приём иностранных туристов в России

## Критические зависимости

```
                    ┌─────────────────┐
                    │     L1-07 ИТ    │
                    │    САМО-тур     │
                    └────────┬────────┘
                             │
             ┌───────────────┼───────────────┐
             ▼               ▼               ▼
        ┌─────────┐    ┌─────────┐    ┌─────────┐
        │ L1-01   │    │ L1-05   │    │ L1-08   │
        │Агентства│    │Операции │    │Финансы  │
        └─────────┘    └─────────┘    └─────────┘
```

**Узкое место:** Все операционные процессы зависят от системы САМО-тур.
**Риск:** Сбой САМО-тур = остановка продаж и бронирований.
**Рекомендация:** Зафиксировать SLA* с поставщиком (см. замечание #7)

*SLA (Service Level Agreement) — соглашение об уровне сервиса: гарантии по доступности, времени реакции на сбои и т.д.

## Словарь терминов

| Термин | Расшифровка |
|--------|-------------|
| B2B | Business-to-Business — продажи юридическим лицам (компаниям) |
| B2C | Business-to-Consumer — продажи физическим лицам (туристам) |
| MICE | Meetings, Incentives, Conferences, Exhibitions — корпоративный туризм |
| LTV | Lifetime Value — пожизненная ценность клиента |
| NPS | Net Promoter Score — индекс лояльности клиентов |
| FCR | First Contact Resolution — решение вопроса с первого обращения |
| SLA | Service Level Agreement — соглашение об уровне сервиса |
| HHI | Herfindahl-Hirschman Index — индекс концентрации (диверсификации) |
| ROS | Return on Sales — рентабельность продаж |
| п.п. | Процентный пункт — единица измерения изменения процентов |
