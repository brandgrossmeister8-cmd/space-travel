## Детальная классификация

### 1. Генераторы выручки

Департаменты, напрямую участвующие в создании и продаже турпродукта. Без них компания не зарабатывает.

---

#### L1-01 — Департамент развития взаимодействия с агентствами

| Параметр | Значение |
|----------|----------|
| **Тип** | Генератор выручки |
| **Роль** | Продажи через партнёрскую сеть |
| **Штат** | 72 чел. (34% компании) |
| **Целевая выручка** | 2,5 → 2,875 млрд руб. (+15%) |
| **Ключевые задачи** | Привлечь 200 агентов, снизить долги на 40% |
| **Критичность** | ВЫСОКАЯ — основной канал продаж |

**Почему генератор выручки:** Обеспечивает ~85% выручки компании. Каждый новый агент — это дополнительные продажи без затрат на собственные офисы.

---

#### L1-02 — Департамент по управлению операционными доходами

| Параметр | Значение |
|----------|----------|
| **Тип** | Генератор выручки |
| **Роль** | Закупки и формирование турпродукта |
| **Штат** | 21 чел. (10% компании) |
| **Целевая маржа** | 12% → 14% (+2 процентных пункта) |
| **Ключевые задачи** | Диверсификация продукта, ускорение вывода новинок |
| **Критичность** | ВЫСОКАЯ — формирует то, что продаётся |

**Почему генератор выручки:** Договаривается с отелями и авиакомпаниями о ценах. От качества переговоров зависит маржа (разница между закупкой и продажей).

**Что такое маржа:** Если закупили тур за 80 000 руб., продали за 100 000 руб. — маржа 20 000 руб. (20%). Цель — увеличить её с 12% до 14%.

---

#### L1-03 — Департамент въездного туризма, MICE и корпоративного обслуживания

| Параметр | Значение |
|----------|----------|
| **Тип** | Генератор выручки |
| **Роль** | Корпоративные продажи (B2B) |
| **Штат** | 6 чел. (3% компании) |
| **Целевая выручка** | 150 → 195 млн руб. (+30%) |
| **Ключевые задачи** | 50 новых корп. клиентов, 20 тендеров |
| **Критичность** | СРЕДНЯЯ — диверсификация бизнеса |

**Почему генератор выручки:** Прямые контракты с компаниями на корпоративные поездки. Один клиент может принести миллионы рублей в год.

**Что такое MICE:**
- **M**eetings — корпоративные встречи и совещания
- **I**ncentives — поощрительные поездки для сотрудников
- **C**onferences — конференции
- **E**xhibitions — выставки и ярмарки

---

#### L1-05 — Операционный департамент

| Параметр | Значение |
|----------|----------|
| **Тип** | Генератор выручки |
| **Роль** | Исполнение заказов |
| **Штат** | 50 чел. (24% компании) |
| **Целевая конверсия** | 25% → 30% (+5 процентных пунктов) |
| **Ключевые задачи** | Ответ за 2 часа, допродажи 12% |
| **Критичность** | ВЫСОКАЯ — точка преобразования лидов в деньги |

**Почему генератор выручки:** Каждый процент конверсии — это дополнительная выручка. Если из 100 обращений покупали 25, а станут покупать 30 — это +20% к выручке при тех же затратах на привлечение.

**Что такое конверсия:** Доля обращений, которые превращаются в оплаченные заказы. Обратились 100 человек, купили 25 = конверсия 25%.

---

### 2. Усилители выручки

Департаменты, увеличивающие выручку через привлечение новых и удержание существующих клиентов.

---

#### L1-06 — Управление клиентского сервиса

| Параметр | Значение |
|----------|----------|
| **Тип** | Усилитель выручки |
| **Роль** | Удержание клиентов |
| **Штат** | 12 чел. (6% компании) |
| **Целевые повторные покупки** | 20% → 35% (+15 процентных пунктов) |
| **Целевой рост LTV** | +40% |
| **Критичность** | ВЫСОКАЯ — удержание дешевле привлечения |

**Почему усилитель выручки:** Не продаёт напрямую, но влияет на то, вернётся ли клиент. Привлечь нового клиента стоит в 5-7 раз дороже, чем удержать существующего.

**Что такое LTV (пожизненная ценность клиента):** Сколько денег клиент принесёт за всё время сотрудничества. Если клиент покупает тур раз в год по 150 000 руб. и остаётся 5 лет — его LTV = 750 000 руб.

**Что такое NPS (индекс лояльности):** Готовность рекомендовать компанию друзьям. Шкала от -100 до +100. Цель Space Travel: +50 (отличный показатель).

---

#### L1-09 — Департамент маркетинга и рекламы

| Параметр | Значение |
|----------|----------|
| **Тип** | Усилитель выручки |
| **Роль** | Привлечение клиентов |
| **Штат** | 18 чел. (9% компании) |
| **Целевая узнаваемость** | +25% среди агентов |
| **Целевой рост партнёрских бюджетов** | +20% |
| **Критичность** | ВЫСОКАЯ (но позиция директора вакантна!) |

**Почему усилитель выручки:** Создаёт поток потенциальных клиентов (лиды). Без маркетинга — нет притока новых клиентов, продажи падают.

**РИСК:** Вакансия директора департамента снижает эффективность. Требуется срочное назначение.

---

### 3. Обеспечивающие функции

Департаменты, обеспечивающие инфраструктуру и поддержку основной деятельности. Сами не зарабатывают, но без них бизнес остановится.

---

#### L1-04 — Департамент управления персоналом (HR)

| Параметр | Значение |
|----------|----------|
| **Тип** | Обеспечивающая функция |
| **Роль** | Кадры |
| **Штат** | 5 чел. (2% компании) |
| **Соотношение** | 1 HR на 42 сотрудника |
| **Целевая укомплектованность** | 95% |
| **Целевая текучесть** | 18% (сейчас ~25%) |

**Почему обеспечивающая функция:** HR нанимает и обучает людей для других департаментов. Сам не продаёт, но без людей продавать некому.

**Что такое текучесть:** Доля сотрудников, уволившихся за год. 25% означает, что каждый год уходит четверть штата. Цель — снизить до 18%.

---

#### L1-07 — Департамент информационных технологий

| Параметр | Значение |
|----------|----------|
| **Тип** | Обеспечивающая функция |
| **Роль** | ИТ-инфраструктура |
| **Штат** | 7 чел. (3% компании) |
| **Соотношение** | 1 IT на 30 сотрудников |
| **Целевая доступность систем** | 99,9% |
| **Целевые потери от сбоев** | 0 руб. |

**Почему обеспечивающая функция:** IT поддерживает системы, без которых невозможны продажи и бронирования. Сам не продаёт, но его сбои = остановка бизнеса.

**РИСК:** Критическая зависимость от внешней системы САМО-тур без формального SLA (соглашения об уровне сервиса).

**Что такое доступность 99,9%:** Система может быть недоступна не более 8,76 часов в год. Остальные 8751 час — работает.

---

#### L1-08 — Финансовый департамент

| Параметр | Значение |
|----------|----------|
| **Тип** | Обеспечивающая функция |
| **Роль** | Деньги и отчётность |
| **Штат** | 20 чел. (9% компании) |
| **Целевая ликвидность** | ≥1,2 |
| **Целевая рентабельность** | ≥5% |
| **Целевой срок закрытия месяца** | 5 дней |

**Почему обеспечивающая функция:** Финансы контролируют денежные потоки, платят поставщикам и зарплаты. Сами не зарабатывают, но без них бизнес рухнет.

**Что такое ликвидность 1,2:** На каждый рубль краткосрочных долгов есть 1,2 рубля для их погашения. Меньше 1,0 = риск неплатежей.

**Что такое рентабельность продаж (ROS):** Сколько прибыли остаётся с каждого рубля выручки. 5% означает: продали на 100 млн руб. — заработали 5 млн руб. чистыми.

## Словарь терминов

| Термин | Расшифровка |
|--------|-------------|
| B2B | Business-to-Business — продажи компаниям |
| B2C | Business-to-Consumer — продажи физлицам |
| MICE | Meetings, Incentives, Conferences, Exhibitions — корпоративный туризм |
| LTV | Lifetime Value — пожизненная ценность клиента |
| NPS | Net Promoter Score — индекс лояльности (готовность рекомендовать) |
| FCR | First Contact Resolution — решение вопроса с первого обращения |
| SLA | Service Level Agreement — соглашение об уровне сервиса |
| HHI | Herfindahl-Hirschman Index — индекс концентрации продуктов |
| ROS | Return on Sales — рентабельность продаж |
| HR | Human Resources — управление персоналом |
| ИТ | Информационные технологии |
| п.п. | Процентный пункт — единица изменения процентов |
| Конверсия | Доля обращений, ставших покупками |
| Маржа | Разница между ценой закупки и продажи |
| Текучесть | Доля уволившихся за год |
| Ликвидность | Способность платить по долгам |
