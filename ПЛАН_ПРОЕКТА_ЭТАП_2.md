## ОБЗОР ПРОЕКТА

### Цель проекта
Провести комплексный анализ организационной структуры, целей, задач и бизнес-процессов компании Space Travel с созданием:
- Сквозной иерархии целей и задач (L1 → L2 → L3)
- Описания бизнес-процессов с привязкой к задачам
- Регламентов по ключевым процессам
- Должностных инструкций с KPI на каждого сотрудника
- **Полного анализа IT-ландшафта и интеграций**
- **Выявления всех точек автоматизации с расчётом ROI**
- **Технических заданий на автоматизацию**
- Дорожной карты автоматизации

### Ключевые принципы
1. **Иерархичность**: Цель департамента → Задачи департамента → Цель отдела → Задачи отдела → Процессы → Функционал сотрудника
2. **SMART-цели**: Все цели должны быть конкретными, измеримыми, достижимыми, релевантными, ограниченными во времени
3. **Уникальность задач**: Каждая задача направлена на достижение конкретной цели, дублирование исключено
4. **Сквозная связность**: Непрерывная цепочка от стратегии до операций
5. **KPI на всех уровнях**: Измеримые показатели для департаментов, отделов и сотрудников
6. **Исключение дублей**: Инструментов, данных, процессов, функций
7. **Прозрачность**: Все связи между подразделениями задокументированы и визуализированы

## ЦЕЛЕВАЯ СТРУКТУРА ДОКУМЕНТОВ

### Иерархия целей-задач-процессов-функций

```
ЦЕЛЬ ДЕПАРТАМЕНТА (SMART) → KPI департамента
    │
    ▼
ЗАДАЧИ ДЕПАРТАМЕНТА (уникальные, направлены на цель)
    │
    ▼
ЦЕЛЬ ОТДЕЛА (SMART, вытекает из цели департамента) → KPI отдела
    │
    ▼
ЗАДАЧИ ОТДЕЛА (коррелируют с задачами департамента)
    │
    ▼
ПРОЦЕССЫ (обеспечивают решение задач)
    │
    ▼
ЭТАПЫ ПРОЦЕССОВ
    │
    ▼
ФУНКЦИОНАЛ СОТРУДНИКА → KPI сотрудника → ДОЛЖНОСТНАЯ ИНСТРУКЦИЯ
```
