# Space Travel: Анализ и оптимизация бизнес-процессов

Проект по комплексному анализу организационной структуры, целей, задач и бизнес-процессов туристической компании Space Travel.

## Цели проекта

- Создание сквозной иерархии целей и задач (L1 → L2 → L3)
- Описание бизнес-процессов с привязкой к задачам
- Разработка регламентов по ключевым процессам
- Формирование должностных инструкций с KPI
- Анализ IT-ландшафта и выявление точек автоматизации

## Структура проекта

```
Space Travel/
├── L1/                    # Данные руководителей департаментов (8 файлов)
├── L1_json/               # Нормализованные данные L1 в JSON
├── L2/                    # Данные руководителей отделов (34 файла)
├── L2_json/               # Нормализованные данные L2 в JSON
├── L3/                    # Данные сотрудников (~150 файлов)
├── L3_json/               # Нормализованные данные L3 в JSON

```
