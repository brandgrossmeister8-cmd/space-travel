# Space Travel: Анализ и оптимизация бизнес-процессов

Проект по комплексному анализу организационной структуры, целей, задач и бизнес-процессов туроператора Space Travel.

## Компания

- **9 департаментов**, 39 L2, 198 сотрудников
- **30 SMART-целей** → **52 задачи** → **1 394 функции**
- **8 бизнес-процессов**, 69 регламентов
- **62 точки автоматизации** → экономия ~14,8 млн ₽/год
